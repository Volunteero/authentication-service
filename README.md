### Local Deployment
* ``git clone git@github.com:Volunteroo/AuthenticationBackend.git``
* ``cd AuthenticationBackend``
* ``copy the .env file to the root directory``
* ``npm install``
* ``npm run dev``
* Service available at localhost:3000
